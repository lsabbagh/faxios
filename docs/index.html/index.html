alias

alias(target: string, name: string, key: string): object
Parameters
target (string) the type of the alias
name (string) the name of the alias @example: .name('Wassim')
key (string = name) the key to be set @example: {firstName: 'Wassim'}
Returns
object: self
Example
faxios()
 .alias('')
 .baseURL('http://jsonplaceholder.typicode.com')
 .alias('param', 'name', 'firstName') // <-- setting the alias, very useful with builders
 .name('Wassim)

  // => .param('firstName', 'Wassim')

 .FETCH // => Promise
 .then(res => {})
 .catch(err => {});
